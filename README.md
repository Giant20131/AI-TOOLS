# AI Tools and Resources Catalog

---

## 1. Primary AI Platforms, Models & Research Tools
* **Anthropic / Claude:** Advanced LLM family known for deep reasoning, coding capabilities, and text generation.
* **ChatGPT (OpenAI):** General-purpose conversational AI and analysis platform.
* **Gemini (Google):** Google's multimodal AI ecosystem.
* **Google AI Studio:** Developer platform for prototyping with Gemini models.
* **Llama (Meta):** Open-weights foundational LLM framework.
* **Ollama:** Local infrastructure for running open-source LLMs efficiently.
* **OpenRouter AI:** Unified API access interface to benchmark and use various open and closed LLMs.
* **Artificial Analysis (`artificialanalysis.ai/models`):** Benchmarking and performance analysis hub for AI models.
* **Prompting Guide (`promptingguide.ai`):** Educational resource and documentation for advanced prompt engineering.
* **Hugging Face:** Central repository and community hub for machine learning models, datasets, and web apps.

---

## 2. Web Design, Code Generation, Automation & 3D generation
* **Stitch:** UI/UX tool designed for prototyping website and app interfaces, with features to export designs straight into an AI studio to compile or generate code.
* **v0.3 (V0 by Vercel):** Generative UI system producing production-ready frontend code from text descriptions.
* **Bolt / Goose:** AI-native development assistants and coding agents designed to build full-stack applications.
* **Lovable:** Full-stack AI software development engine that turns prompts into web applications.
* **Make.com:** Visual workflow automation and integration platform connecting various web applications.
* **Zapier:** Popular workflow automation platform designed to link apps and automate manual workflows.
* **Apify:** Web scraping and data extraction platform built to feed data into AI workflows.
* **Tripo 3D:** AI-powered platform that converts text descriptions, images, or sketches into detailed 3D models within minutes.

## 3. Productivity, Knowledge & Data Management
* **Notion:** All-in-one workspace for notes, documentation, project planning, and database organization.
* **Glean AI:** Enterprise search engine powered by AI to seamlessly search internal company apps and documents.
* **Linear:** Streamlined project tracking and issue management tool for modern software teams.
* **NotebookLM (Google):** Personalized AI research assistant built around specialized uploaded documents.
* **Numerous.ai:** AI integration for spreadsheets (Google Sheets / Excel) to automate data categorization and formula writing.
* **Perplexity / Comet (Perplexity Browser Extension):** Conversational search engine and browser companion providing cited, real-time answers.
* **Grok:** X-integrated (Twitter) AI search engine focused on social media data streams and real-time public sentiment.

## 4. Audio, Video & Creative Media
* **Suno:** Generative AI music platform capable of creating complete vocal and instrumental tracks from text prompts.
* **ElevenLabs:** Realistic text-to-speech, voice cloning, and audio generation tools.
* **HeyGen:** AI video generation platform focusing on localized digital avatars and automatic video translation.
* **Fireflies.ai:** Automated meeting recorder, transcription engine, and conversation intelligence platform.
* **Happy Scribe:** Audio/video transcription and subtitle generation service.
* **Freepik / Krea / Phot.ai:** Digital creative suites emphasizing AI image generation, vector enhancement, and asset creation.
* **Final Cut Pro / DaVinci:** Professional, industry-standard video editing, color correction, and post-production software.
* **Gamma:** AI-assisted document, webpage, and presentation deck generation platform.
